Managing a user's profile


The User profile API helps you create and manage archives.

The base resource for this API is https://{{ request.META.HTTP_HOST }}/api/v1/user/

View a basic user profile

View user account details.

curl {{ request.META.HTTP_HOST }}/api/v1/user/?api_key=your-api-key

This will display the basic information associated with a user's account. Pass in your API key using a GET.

{"first_name": "Regular", "full_name": "Regular User", "groups": [{"name": "user"}], "id": 4, "last_name": "User", "short_name": "Regular"}
