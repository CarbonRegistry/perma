Perma Link Creation

Creating a Perma Link

A Perma Link gives you access to an archived version of the content at a URL, frozen at a moment in time. To create a Perma Link, simply input a URL into the entry box, as shown below, and click on the blue infinity button!



The entry box is flexible and will accept multiple formats, including www and https://.

After the link is processed, a thumbnail of the original webpage will appear, now linked to a Perma Link address. The option allows you to easily copy the Perma Link for inclusion in a document, or testing on another page.

At this point, you have created an unvested, NON-PERMANENT link.

If your website is not correctly rendering using Perma’s standard toolkit, use the upload button to upload screenshots (.png, .jpg) or PDF documents. The upload option appears at the bottom of the website thumbnail, immediately following creation of the Perma Link.

To see all URL’s that you have converted to Perma Links, click on Links in the navigation menu on the left.


Organizing Links

Perma includes many useful tools for organization of links. You can file links in folders. Change the titles of your links. Even add notes that can be viewed by users who create or vest links.

You can search or sort your links using the search bar above the link list or clicking the column headings.
