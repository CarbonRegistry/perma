Perma Dev

API Key

All API calls require the api_key parameter. Log into your Linky account to create or retrieve you API key.

Format

All calls default to json unless otherwise specifed. You can select your prefered output with the format parameter (can be json or xml).

Create

POST to http://perma.cc/api/v1/link/

Creates new Perma, returns identifier.

Parameters:

url, title, (file)

Example:
curl -F "api_key=RHRMTOQSIF6F87Z11" -F "format=json" -F "title=Google" -F "url=example.com" "http://perma.cc/api/v1/link/"
{ "status" : "success", "response" : { "linky_id": "08uLuvwM3Fh", "linky_hash": "08uLuvwM3Fh" } }

Read

GET to http://perma.cc/api/v1/link/

Returns metadata and URLs to archives. Returns status of indexing jobs too.

Parameters:

link_id

Example:
http://perma.cc/api/v1/link?api_key=RHRMTOQSIF6F87Z11&format=json&linky_id=08uLuvwM3Fh
{ "status" : "success", "response" : { "linky_id" : "08uLuvwM3Fh", "title" : "Welcome to example.com!", "url" : "http://example.com", "body" : "Hi!", "img_cap" : "http://perma.law.harvard.edu/static/2013/10/1/21/44/08RxzoS4tSw/cap.png" } }

Vest

POST to http://perma.cc/api/v1/link/vest

Vests an existing Perma link. Your accout must have vesting privlidges to make this call.

Parameters:

linky_id

Example:
http://perma.cc/api/v1/link/vest?api_key=RHRMTOQSIF6F87Z11&format=json&linky_id=08uLuvwM3Fh
{ "status" : "success", "response" : { "linky_id" : "08uLuvwM3Fh", "title" : "Welcome to example.com!", "url" : "http://example.com", } }
