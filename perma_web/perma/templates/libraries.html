Libraries power Perma.cc

For centuries, libraries have worked together to preserve print sources and to make those sources accessible to the public. Today they do the same for online sources through Perma’s active network of library partners.

How Perma.cc works for libraries

We call our library partners “registrars.” Each registrar sponsors and administers Perma for its local community of scholars and journals. Check out our user guide to learn more about how libraries use Perma.cc.
